General Configuration

  • s!config do<NSFW Category> <0|1>
    enable(1) or disable(0) a category of NSFW command.
    e.g. s!config doShark 0
  • s!config doGay <0|1>
    enable(1) or disable(0) gay variants of any commands.
    e.g. s!config doGay 1
  • s!config scalieFeedChannel <channel_id>
    Set a channel to become a scalieFeedChannel.
    This will cause the channel to receive new images every ten minutes.
    e.g. s!config scalieFeedChannel 1001598070468915271
  • s!config scalieVoteChannel <channel_id>
    Set a channel to become a scalieVoteChannel.
    This will cause the channel to receive the image with the most votes per day.
    e.g. s!config scalieVoteChannel 1001598070468915271
  • s!config onlyNsfwChannels <0|1>
    enabled(1) or disable(0) restricting NSFW commands to NSFW channels.
    e.g. s!config onlyNsfwChannels 1

Configure your server to be found via s!find

To do this, you must have correctly set every variable needed. Run each of the following commands:

s!config enabled 1
s!config title <your server name here>
s!config description <a brief server description>
s!config invite <your invite link>
s!config image_url <url link to an image>

If all commands have been run successfully, you will appear on the s!find command.
